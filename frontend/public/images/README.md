# Images for Autism Module

Place your images in this folder with the exact names below.

## Required Image Files:

### Tamil Lesson
- `tamil-greeting.png` or `.jpg` - Image representing greeting/hello
- `tamil-thanks.png` or `.jpg` - Image representing gratitude/thank you
- `tamil-goodbye.png` or `.jpg` - Image representing parting/goodbye

### English Lesson
- `letter-a.png` or `.jpg` - Image for letter A (e.g., Apple)
- `letter-b.png` or `.jpg` - Image for letter B (e.g., Ball)
- `letter-c.png` or `.jpg` - Image for letter C (e.g., Cat)

### Hindi Lesson
- `hindi-one.png` or `.jpg` - Image representing number 1
- `hindi-two.png` or `.jpg` - Image representing number 2
- `hindi-three.png` or `.jpg` - Image representing number 3

## Image Requirements:
- Format: PNG or JPG
- Recommended size: 300x300 pixels (square)
- Simple, uncluttered visuals
- Appropriate for all ages
- Clear, high contrast

## How to Add Images:
1. Prepare your images with the exact names above
2. Place them in this `/public/images/` folder
3. Refresh your browser - images will appear automatically!

## Note:
If images are not found, colorful emoji icons are shown as fallback.
